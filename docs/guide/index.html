第 0 章 · 系統導覽

系統簡介

排班系統是網頁版班表管理工具，排班、請假、加班皆需登入帳號後才能使用。主要功能包含：

  • 班表編輯 — 八週週期排班，支援人員與班別兩種檢視
  • 班別 / 假別 / 加班設定 — 型錄維護與工具列 chip 快速填入
  • 人員與單位管理 — 主檔維護、排班單位優先序
  • 休假與加班申請 — 員工送件、主管審核
  • 例休檢查 — 例假、休息日、連續出勤合規驗證
  • 匯入 / 匯出 — Excel 主檔、SAP 休例假 CSV
自動排班演算法尚未上線，目前僅完成前置設定欄位，供日後擴充使用。

第 0 章 · 系統導覽

登入與帳號

開啟班表網址後，系統會先顯示登入畫面，完成登入後才能進入班表與各項功能。

  1. 輸入工號與密碼
  2. 登入成功後進入班表，頂部列顯示姓名與對應功能按鈕
  3. 依帳號權限顯示員工或主管功能
登入使用工號，非 email 地址。新建立帳號的預設密碼為 0000，首次登入後建議立即修改密碼。
若忘記密碼，請聯繫主管透過「人員設定」重設為預設密碼 0000。
第 1 章 · 員工

登入與登出

  1. 開啟網址後在登入畫面輸入工號（例如 A001）與密碼
  2. 登入後頂部顯示姓名，並出現員工功能按鈕
新帳號預設密碼為 0000。登入後請儘快使用「修改密碼」變更。

登出：點選頂部列「登出」，系統清除 session，需重新登入才能繼續使用。

第 1 章 · 員工

檢視個人班表

員工登入後進入唯讀模式（班表主頁）：

  • 可瀏覽班表、切換週期與檢視模式
  • 工具列 chip 為唯讀，無法點選編輯格
  • 無「功能」選單與各項設定入口

申請狀態 overlay：

  • 待審核申請 — 以特殊配色顯示在對應日期格
  • 已核准申請 — 自動寫入班表並顯示假別 / 加班標記
  • 已退回申請 — 不顯示於班表，可重新送件
第 1 章 · 員工

請假申請

  1. 點選頂部列「請假申請」
  2. 選擇假別（例假、休息日、特休等）
  3. 設定日期區間、整天或時段、原因（若該假別需要）
  4. 按「送出」提交申請
送出後申請狀態為「待審核」，對應班表格會被鎖定，需等主管審核或自行刪除待審核申請後才能變更。
第 1 章 · 員工

管理我的請假

在「請假申請」視窗中可查看所有個人申請：

  • 待審核 — 可刪除後重新送件
  • 已核准 — 已寫入班表，無法自行刪除
  • 已退回 — 可查看主管備註，必要時重新申請
僅待審核狀態的申請可由員工自行刪除。已核准或已退回需聯繫主管處理。
第 1 章 · 員工

加班申請

  1. 點選頂部列「加班申請」
  2. 選擇加班類型
  3. 設定日期、起迄時段
  4. 填寫休息 1、休息 2 時段（若適用）
  5. 填寫原因並送出

審核流程與請假相同：待審核 → 主管核准 / 退回 → 核准後寫入班表。

第 1 章 · 員工

管理我的加班

在「加班申請」視窗中管理個人加班紀錄，操作方式與請假相同：

  • 查看各筆申請的審核狀態與主管備註
  • 刪除待審核的申請
  • 已核准的加班會顯示在班表對應日期格
第 1 章 · 員工

修改密碼

  1. 點選頂部列「修改密碼」
  2. 輸入目前密碼與新密碼
  3. 確認新密碼後儲存
若忘記密碼，請聯繫主管透過「人員設定」重設為預設密碼 0000。
第 1 章 · 員工

員工注意事項

  • 員工無法直接編輯班表格，所有排班變更需透過申請由主管審核
  • 有待審核申請的日期格會被鎖定，需先刪除或等審核完成
  • 主管核准後，假別 / 加班會自動寫入班表
  • 一人一天最多一班（系統規則，自動排班時亦適用）
  • 空白未排班不會自動視為例假或休息日
第 2 章 · 主管

主管功能總覽

頂部「功能」選單

  • 請假審核
  • 加班審核
  • 週期設定
  • 匯出休例假 / 請假 / 加班

工具列與表頭

  • 班別 / 假別 / 加班 chip 編輯
  • 例休檢查
  • 各型錄設定
  • 表頭 → 單位設定 / 人員設定
主管登入後工具列完整展開，可直接編輯排班格與所有主檔設定。
第 2 章 · 主管

排班編輯基礎

  1. 在工具列選取班別 / 假別 / 加班 chip（chip 會高亮）
  2. 點選班表上的目標格，內容即寫入
  3. 框選多格：拖曳選取範圍，批次套用目前 chip
  4. 再次點選已選 chip 或按清除，可取消選取
  5. 假別 / 加班可能需要填寫時段與原因（彈窗）
若該格有員工待審核申請，需先審核或退回後才能修改。
第 2 章 · 主管

檢視模式

人員檢視

  • 單位 × 人員 × 日期
  • 適合逐人排班
  • 可拖曳調整人員單位歸屬

班別檢視

  • 班別 × 需求人數 × 日期
  • 顯示當日已排人員
  • 缺額以標示提醒（依「需求人數」）
「需求人數」在班別設定中維護，是日後自動排班補人的基礎資料。
第 2 章 · 主管

直接在班表指定假 / 加班

主管在班表上直接填入假別或加班時：

  • 系統自動建立已核准紀錄（source = manager）
  • 無需再進審核流程
  • 填寫時段、原因等明細後即寫入班表
手動指定的班與假，在自動排班規劃中視為鎖定，演算法不會覆蓋。
第 2 章 · 主管

單位設定

入口：班表表頭「單位」欄的設定圖示（班表主頁）

  • 單位檢視 — 各單位的所屬人員、可排人員
  • 人員檢視 — 各人員的排班單位排序
  • 新增 / 修改 / 刪除單位（名稱、有效期間）
  • 拖曳調整人員單位歸屬
  • Excel 匯出 / 匯入單位主檔
第 2 章 · 主管

人員設定

入口：班表表頭「人員」欄的設定圖示（班表主頁）

  • 工號、姓名、權限（員工 / 主管）
  • 到職日、離職日、計薪方式
  • 排班單位 — 可勾選多單位並拖曳排序，第 1 個為主要單位
  • 月休天數 — 該月例假 + 休息日的目標天數
  • 例假星期 — 固定例假偏好
  • 重設密碼為 0000（建立新登入帳號時亦使用預設密碼）
  • Excel 匯出 / 匯入人員主檔
第 2 章 · 主管

班別 / 假別 / 加班設定

各工具列區塊的設定按鈕（鉛筆圖示）進入型錄維護（班表主頁）：

型錄重點欄位
班別顏色、適用單位、時段、需求人數、拖曳排序
假別顏色、SAP 代碼、是否需填時間/原因、申請顯示色
加班顏色、預設時段、休息 1/2、申請顯示色

各型錄均支援 Excel 匯出 / 匯入。

第 2 章 · 主管

週期設定

入口：功能選單 → 週期設定

  • 八週起算日 — 班表預設顯示的 56 天週期錨點
  • 每週起算日 — 週界線著色、例休檢查的週切分基準
  • 每月起算日 — 月休天數計算與自動排班月區間基準
修改週期設定後，班表格線與例休檢查結果會即時連動更新。
第 2 章 · 主管

請假審核

入口：功能選單 → 請假審核（或格 tooltip 的「審核」捷徑）

  1. 依申請人、日期、狀態篩選待審清單
  2. 選擇「核准」或「退回」，可填寫主管備註
  3. 核准後系統自動將假別寫入班表
員工申請（source = employee）需經此審核；主管直接在班表指定的假別則已視為核准。
第 2 章 · 主管

加班審核

操作方式與請假審核相同：

  1. 功能選單 → 加班審核
  2. 篩選並檢視待審加班申請
  3. 核准或退回，填寫備註
  4. 核准後寫入班表對應格的加班標記
第 2 章 · 主管

例休檢查

入口：工具列假別區 → 例休檢查

  • 每 7 天至少 1 天例假（SAP 0036）
  • 每 7 天至少 1 天休息日（SAP 0047）
  • 連續出勤不得超過 6 天 — 任意滑動視窗，含上月銜接
  • 到職 / 離職週有特殊略過或改檢查規則
  • 例假日有排班或加班會產生警告
檢查以已標記的例假 / 休息日為準；空白格不自動視為例假或休息日。
第 2 章 · 主管

匯出作業

入口：功能選單

匯出項目格式說明

附錄

自動排班（規劃中）與 FAQ

已就緒的前置欄位（演算法尚未上線）：

  • 班別「需求人數」、人員「排班單位優先序」（人員設定）
  • 人員「月休天數」、週期「每週 / 每月起算日」（週期設定）

常見 FAQ：

  • 為何無法儲存？ — 儲存班表需主管權限
  • 格被鎖定？ — 該日有待審核員工申請
  • 忘記密碼？ — 請主管在人員設定重設為 0000
  • 新進人員如何登入？ — 主管新增人員後自動建立帳號，預設密碼 0000
